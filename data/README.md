# Data

- `hand_histories/raw/` original hand history files
- `hand_histories/summaries/` optional tournament summary sidecars used for field-size-aware stage notes, including imported `tournament_<id>__<label>.txt` files from `code/scripts/import_tournament_summaries.py`
- `hand_histories/parsed/` parsed outputs
- `hand_histories/metadata/` optional PKO sidecar metadata used for bounty EV conversion in bounded preflop spots
- `datasets/` cleaned datasets
- `reference/` supporting files, charts, lookup tables
